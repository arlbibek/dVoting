# Decentralized Voting (dVoting)

A decentralized voting system based on Ethereum blockchain technology for corporate use.

## Requirments

### Node.js

> Node.js v14.15.4

Download Node.js from [here](https://nodejs.org/en/download/).

#### Truffle

> truffle@5.2.2

```shell
npm install -g truffle # installing truffle via node package manager (npm)

mkdir dVoting
cd dVoting
truffle init # initializing the created project directory with the Truffle
```
